# AIForge
Simple library for LLM token accounting

> “First principles: break problems down to fundamental truths and build up from there.”

---

## Overview

AIForge is a minimal developer toolkit designed to help builders work with **AI systems and large language models** more efficiently.

The focus is simple:

- fast experimentation
- lightweight utilities
- minimal complexity
- practical tools for developers

AIForge aims to make working with AI models easier by providing small, focused tools instead of heavy frameworks.

---

## Features

- ⚡ Lightweight AI utilities  
- 🤖 LLM experimentation tools  
- 🧮 Token and cost calculations  
- 🧩 Modular architecture  
- 🛠 Developer-friendly design  

---

## Tech Stack

- TypeScript
- Node.js
- CLI tools
- Web utilities
